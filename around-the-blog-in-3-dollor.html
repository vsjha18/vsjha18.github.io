Around The Blog in 3 dollor !!

Mon 26 January 2015

From a long time I was looking for a cheap alternative to blogging which doesn't require me go through complex and costly hosting service. The main reason was, I was never sure whether I will be able to write even my second article.

There were several options but not was the perfect one for me. requirements were as follows:

  • Should have less or no hosting hasles.
  • Doesn't require me to learn a different language, just for this thing.
  • I should be able to style it with basic CSS skills.
  • I should be able to write the content on plain text editor like vim.

Wordpress

The holy grail of all blogging frameworks. I did't like it becuase, first you require to buy a hosting service to get the full control. Secondly you may also require to learn PHP in case you want to go further.

Google

This is free and only cost you may incurr will be possibly will be of domain name. But the main problem is the limitation on themes and it is very limited in the option to which it can be configured. But for me still it was better than Wrodpress.

Pinky says this is neat but not great !

Now Comes the Static Site Generators

This is an amazing solution for casual bloggers. The biggest boon is that you can actually get rid of the server side. You just need a server which can host your static files.

LinkedIn

Linked is a great website for connecting to right people. I also have an account.
